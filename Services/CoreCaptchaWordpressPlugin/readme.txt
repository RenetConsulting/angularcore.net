Adds a clean audio-based captcha to Contact Form 7

== Description ==

This CAPTCHA is used in combination with audio-based. Audio CAPTCHAs present an audio recording of a series of letters or numbers which a user then enters.

= Directions [PLEASE READ] =
Add the shortcode [core-captcha] to the form editor where you want the captcha to appear.

== Installation ==

1. Upload contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter the valid parameters through the 'Settings' -> 'Core captcha' menu in WordPress
3. Place `[core-captcha]` to your contact form 7 forms
